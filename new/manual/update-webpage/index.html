Using the update webpage

To access the webpage, follow the instructions of the Update>Manual menu.

Wi-Fi​

Here you can configure the details of the Wi-Fi network that will be used for automatic updates in the Update>Automatic menu.

  • Name - the SSID of the Wi-Fi network
  • Password - the password of the Wi-Fi network
  • Set - save the Wi-Fi network details

note

Write the name and password in their respective fields, and press Set.

A message will appear on the page, indicating whether or not the operation was successful.

Configuration​

  • Select .bin file - choose the file from your device
  • Upload - upload the selected configuration file
  • Download - download the currently installed configuration file (for editing)

note

Refer to the Configuration editor chapter to create your configuration file.

After selecting the file and pressing Upload, the progress bar will start to fill up.

At the end, a message will appear on the page, indicating whether or not the operation was successful.

warning

You must wait until you receive a message on the webpage, and you must press OK on it! The system will restart afterwards.

Boot logo​


By default, the boot logo is disabled. Check the Enable boot logo checkbox to enable it.

  • Select .bmp file - choose a bitmap image from your device
  • X coordinate - top-left corner horizontal position on the screen (0-63)
  • Y coordinate - top-left corner vertical position on the screen (0-47)
  • Duration - how many milliseconds the image stays on screen
  • Set - upload the selected boot logo with its settings

Format

Only images with .bmp format are accepted, and 1-bit monochrome is preferred.

Preview

Pressing the Set button will preview the boot logo on the screen.

warning

Please avoid pressing the Set button while the a preview is on the screen!

Removing boot logo

If there is a boot logo uploaded, it can be removed by unchecking the Enable boot logo checkbox. This does not delete the image, and it can be enabled again.

To delete the boot logo, press Delete after disabling it.

Skip

It is possible to skip the boot logo before the duration runs out by using any button (Up/Down/Select).

Update​

  • Select .bin file - choose a software update file from your device
  • Update - install the selected software update

note

After selecting the file and pressing Update, the progress bar will start to fill up.

At the end, a message will appear on the page, indicating whether or not the operation was successful.

warning

You must wait until you receive a message on the webpage, and you must press OK on it! The system will restart afterwards.

warning

Do not turn off the ignition, turn off Wi-Fi or exit the update page while the update is in progress!

More than likely, nothing bad would happen, just that the update wouldn't be installed. But it's recommended to be cautious.

Update failed

If the update failed to install, you will see the regular main menu, without any other messages.

Please try to install the update again.

note

The configuration file, settings, boot logo and other files will not be deleted after the update.

Other files​

  • Please select a file - choose a file from your device
  • Upload - send the selected file

note

After selecting the file and pressing Upload, the progress bar will start to fill up.

At the end, a message will appear on the page, indicating whether or not the operation was successful.

Accepted files

For now, the only accepted files are fault code description files and UDS module files. Check out the Fault code descriptions chapter for more details.

note

You can find every file that uploaded successfully in the Files menu.
